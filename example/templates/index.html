Example project

This project shows example configuration for django-urldecorators application.

The following links point to pages with different decorators/middleware classes applied:

  • Cached page
  • Private page

Example configuration

from urldecorators import patterns, url, include

urlpatterns = patterns('',
    url(r'^private/$', include('example.private.urls'),
        decorators=['django.contrib.auth.decorators.login_required']),
    url(r'^cached/$', include('example.cached.urls'),
        middleware_classes=['django.middleware.cache.CacheMiddleware']),
)
